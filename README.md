# Industry Neutral Portfolio Analysis

This project builds industry-neutral portfolios using stocks and inverse ETFs.

Industries analyzed:
Healthcare (JNJ + RXD)
Energy (XOM + ERY)

The portfolios are constructed so that each portfolio has zero correlation with its industry during the training period.

Methodology:
- Collected monthly price data
- Converted prices to log returns
- Split the sample into training (75%) and test (25%)
- Constructed neutral portfolios using the training data
- Evaluated risk metrics using out-of-sample data
- Compared performance with the Fama-French market factor

Key Skills Demonstrated:
- Portfolio Construction
- Log Return Calculation
- Risk Analysis
- Financial Modeling in Excel
- Quantitative Finance Methods
